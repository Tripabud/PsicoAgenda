# **App Name**: SessionZen

## Core Features:

- Interactive Schedule: Interactive weekly schedule grid with time slots for patient scheduling, attendance tracking, and payment recording.
- Automated Week Management: Automated week creation and data persistence with the ability to copy patient schedules from the current week to the next, excluding attendance and payment data.
- Weekly Report Generation: Generates a weekly report including total scheduled sessions, attended sessions, attendance percentage, total billed, and a list of patients with pending payments.

## Style Guidelines:

- Primary color: Calming Teal (#4DB6AC) to promote a sense of tranquility and focus.
- Secondary color: Soft Gray (#E0E0E0) for a clean and professional background.
- Accent: Light Orange (#FFB300) for highlighting important actions and buttons.
- Clean and structured grid layout for easy navigation.
- Simple, professional icons for actions like 'Attend', 'Payment', and 'Delete'.
- Subtle transitions for navigating between weeks.
